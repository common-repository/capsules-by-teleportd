Embed a live-updating set of social smartphone images curated through your account on http://capsules.teleportd.com.

== Description ==

Capsules is designed to display the content of a capsule created via Teleportd on a WordPress site (you can request a Capsules account at http://teleportd.com).

Features:

- Display the content of a capsule either by clicking the Capsules button in the editor toolbar and referencing its ID or by inserting the shortcode [capsule]ID[/capsule] 
(e.g. [capsule]727adf105c9fb7311df7b670747ffd398e18e6d0[/capsule]).
- [Optional] Set the width and height of the embedded capsule. Default ratio is set to a 500x500 square.

== Installation ==

1. Upload the `teleportd/` directory containing the source of the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy the Teleportd Capsules button in the editor toolbar.
